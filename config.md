
# Environment variables

| Variable Name                      | Description                                                                                                                                                          | Secret |
|------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| `WEB_PORT`                         |                                                                                                                                                                      | false  |
| `NEXT_PUBLIC_APP_DEV_AUTH_URL`     | The backend base url                                                                                                                                                 | false  |
| `CLOUD_NAME`                       | Cloudinary                                                                                                                                                           | true   | 
| `GOOGLE_MAPS_API_KEY`              | Google maps api secret key                                                                                                                                           | true   | 
| `COUNTRY_CODE_URL`                 | For google maps                                                                                                                                                      | false  |
| `APP_DEV_IV_KEY`                   | Encryption  keys                                                                                                                                                     | true   |
| `APP_DEV_IV_ENCRYPTION_SECRET_KEY` | Encryption key                                                                                                                                                       | true   |
| `UPLOAD_PRESET`                    | Name of the Cloudinary preset that defines upload rules (e.g., target folder, allowed formats, transformation options). Used for unsigned uploads from the frontend. | false  |
| `APP_URL`                          | The frontend base url                                                                                                                                                | false  |
